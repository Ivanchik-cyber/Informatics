# Информатикс
Информатикс - олимпиадный проект.
Это платформа для обучения пользованием телефоном и некоторыми сервисами.
Написан на Vue и JS.
Он состоит из 2 частей: сервера и клиента.

> [!CAUTION]
> **Сначала запустите сервер, эатем - клиент.**

> [!IMPORTANT]
> **У вас должен быть установлен [Node и npm](https://nodejs.org/en/).**

Сделано Бурылиным Иваном в команде "Один в поле не воин(".

## Установка сервера
1. Скачайте и распакуйте `Informatics Server.zip`
2. Откройте командную строку
3. Введите команду, которая ведёт в установленную папку
   ``` cmd
   cd {путь к папке}
   ```
4. Введите команду, которая запускает сервер
   ```
   node server.js
   ```
5. Убедитесь, что написано
   ```
   Сервер запущен на порте 8080.
   Успешное подключение к датабазе.
   ``` 

## Установка клиента
1. Скачайте и распакуйте `Informatics Client.zip`
2. Откройте командную строку
3. Введите команду, которая ведёт в установленную папку
   ``` cmd
   cd {путь к папке}
   ```
4. Введите команду, которая запускает клиент
   ```
   npm run dev
   ```
5. Нажмите `o + Enter`

***Удачи!***
